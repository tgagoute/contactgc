Mode d'emploi administrateur

Guide complet de l'application

＋
1. Ajouter / modifier un contact ▾
1
Ouvrir le formulaire
Cliquer sur ＋ Nouveau pour créer ou sur ✏️ pour modifier un contact existant.
2
Type : Résident ou Personnel
Si Personnel est sélectionné, la section Informations médicales est automatiquement masquée.
3
Bouton 👁 Afficher
Le 3e bouton ouvre une fiche de visualisation complète avec le nom en très grand en haut de page.
💡 Seuls les Administrateurs peuvent supprimer des contacts.
👁
2. Fiche de visualisation contact ▾
1
Cliquer sur 👁
S'ouvre une fiche complète avec le Nom et Prénom affiché en très grand caractère en haut sur fond sombre.
2
Sections organisées
Identification, Localisation, Informations médicales (si Résident), Contact et représentation.
3
Accès rapide au formulaire
Un bouton ✏️ Modifier est disponible directement depuis la fiche de visualisation.
🔑
3. Gestion des mots de passe ▾
1
Modifier son propre mot de passe
Cliquer sur ⚙️ Mon compte dans la barre de navigation. Saisir l'ancien mot de passe, puis le nouveau (2 fois).
2
Récupération (écran de connexion)
Cliquer sur Mot de passe oublié ? sur l'écran de connexion. L'administrateur peut voir et réinitialiser les mots de passe.
3
Réinitialisation admin
Dans la page Utilisateurs, cliquer sur 🔑 à côté d'un compte pour voir et modifier son mot de passe.
⚠️ Notez bien les nouveaux mots de passe — ils sont stockés localement sans cryptage avancé.
👥
4. Gestion des utilisateurs (Admin) ▾
1
Créer un compte
Onglet 👥 Utilisateurs visible uniquement par les administrateurs.
2
Rôles
Administrateur : accès complet, gestion comptes, suppression contacts.
Utilisateur : consultation et modification uniquement.
